Introduction

Principal component analysis (or PCA) is a linear transformation method whose main goal is to find the features or components that maximize the variance in a given data-set. In PCA we try to reduce the number of features that we have, in order to try to define the information, in other words, with PCA we reduce the number of independent variables. A typical principal component graph shows two principal components, which are a linear combination of the original variables sorted by the variance.
NOTE: The first principal component (PC1) is the one that shows the largest variance and the PC-th component the lowest variance.

Data files (.csv) info

The interactive visualisation uses hands.csv and hands_pca.csv .
    hands.csv is a dataset of 112 dimensions, containing 40 rows (each of the 40 rows corresponds to one hand outline coordinates). The first 56 variables of each row are x-coordinates of points around the hands outline and the next 56 variables are y-coordinates.
    hands_pca.csv is a file obtained obtained when applying the PCA to hands.csv file. This file contains 40 rows (40 hands) and each column corresponds to a principal component (40 dimensions with variance were generated, the rest are zero).

Instructions of the Visualisation 1

The visualization is contained inside a dotted box.
There are two graphs: the plot of the right shows a scatterplot of the first two principal components of the PCA performed on the hands datasets. The plot on the left shows the outline of one specific hand.
When mousing over one dot of the scatterplot, an orange label will indicate to which hand that PC-point belongs and that point will increase it size, besides that, when double clicking on one dot, the outline of the hand on the left plot will change automatically (with a transition effect).
NOTE: The hand that appears by default when loading the page (in the left graph is hand 0).
In the next section, there are several fragments highlighted in colors that refer to different hands. When mousing over the coloured fragment, the corresponding dot in the scatterplot will be highlighted with the same color as the word fragment.
